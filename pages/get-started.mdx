LYTS is a set of layout primitives to build any kind of layout quickly and confidently. There's nothing to configure, set up or mess up. Components work out-of-the-box with no configuration and sensible defaults.

## Installation

```sh
npm install @christiankaindl/lyts
```

Import the library's CSS styles into your application. For example in a Next.js app, import in `_app.js`:

```jsx
/* _app.js */
import '@christiankaindl/lyts/style.css' // Import the library styles here

function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />
}

export default MyApp
```

That's the whole setup! Now import one of the base components—<Link href='/components/stack'>Stack</Link>, <Link href='/components/row'>Row</Link>, <Link href='/components/clamp'>Clamp</Link>, <Link href='/components/columns'>Columns</Link>, <Link href='/components/grid'>Grid</Link>—and start building great layouts.

## Usage

Compose layout components until you achieve your desired layout. The following \<CenterCard> component creates a card with a max-width of 400px and centers it:

<DebugProvider>
  <Clamp clamp='400px'>
    <Stack gap={0.75} className='card'>
      <h3>Card title</h3>
      <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
    </Stack>
  </Clamp>
</DebugProvider>

```jsx
const CenterCard: FunctionComponent = function () {
  return (
    // A card with clamped 400px and centered
    <Clamp clamp='400px'>
      <Stack gap={0.75} className='card'>
        <h3>Card title</h3>
        <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
      </Stack>
    </Clamp>
  )
}
```
